# event-space-form

イベントスペースの利用申込フォームです。利用規約への同意と申込内容（ニックネーム・利用日程）をサーバー側で Markdown ファイルとして記録します。

## 機能

- 利用規約の表示・同意チェック
- ニックネームと利用日程（開始日・終了日）の入力
- サーバー側で申込内容と規約同意をMarkdownファイルに記録
- 申込完了ページへのリダイレクト

## 技術スタック

- Node.js（標準モジュールのみ、フレームワークなし）
- HTML / CSS / JavaScript（バニラ）

## 使い方

```bash
node server.js
```

起動後、ブラウザで `http://localhost:3000/` にアクセスしてください。

申込データは `submissions/` ディレクトリに Markdown ファイルとして保存されます。
